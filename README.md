# Todo-Application
A simple and neat **Todo Application** which is built using:
1. SpringBoot
2. JPA(Hibernate) + MySQL
3. Lombok
4. Thymeleaf + Bootstrap CSS

This application allows user to:
* Add new tasks
* Mark the tasks as completed (toggle functionality)
* Delete the tasks
